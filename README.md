# PropGuard System - GitHub Pages Starter

This is a front-end starter for **PropGuard System**.

## Files
- `index.html` -> main landing page + dashboard prototype
- `styles.css` -> full visual styling
- `app.js` -> risk calculator + traffic light + quick trade logging

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. GitHub will publish your site.

## Connect to your domain
Because you already own `propguardsystem.com`, add it in:
- **Settings > Pages > Custom domain**

Then update your DNS at your domain provider to point to GitHub Pages.
